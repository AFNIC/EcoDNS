# Architecture

Figure 1 shows the basic building blocks for setting up a Roaming LoRaWAN plateform. The different building blocks involved are:
1.	End-Device (ED)
2.	Radio Gateway (RGW)
3.	Network Server (NS)
4.	Application Server (AS)
5.	Domain Naming System (DNS) Database
